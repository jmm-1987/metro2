Importar Base de Datos SQLite

¡ADVERTENCIA CRÍTICA!

Esta operación REEMPLAZARÁ COMPLETAMENTE la base de datos actual con el archivo SQLite que subas. TODOS los datos actuales se perderán y serán reemplazados por los datos del archivo importado.

Selecciona el archivo .db de SQLite que quieres importar. Este archivo reemplazará completamente la base de datos actual.
Información importante:
  • Se creará automáticamente un respaldo del archivo actual antes de la importación
  • El archivo debe ser una base de datos SQLite válida
  • La aplicación se reiniciará la conexión a la base de datos después de la importación
  • Es recomendable hacer un respaldo manual adicional antes de proceder
Cancelar
Proceso de importación
Pasos que se realizarán:
  1. Se verifica que el archivo sea válido
  2. Se cierra la conexión actual a la base de datos
  3. Se crea un respaldo automático del archivo database.db actual
  4. Se reemplaza el archivo database.db con el archivo importado
  5. Se reinicializa la conexión a la base de datos
  6. Se confirma la importación exitosa
Requisitos del archivo:
  • Debe ser un archivo .db válido de SQLite
  • Debe contener todas las tablas necesarias (usuarios, clientes, tareas, eventos, respuestas_formulario)
  • Debe tener la misma estructura que la base de datos actual
